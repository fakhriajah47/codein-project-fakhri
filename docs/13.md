# 13 - UI Component Spec: Modern Neobrutalism Design System

## 1. Tujuan Dokumen

Dokumen ini menjelaskan design system dan spesifikasi komponen UI untuk **ProjectPilot AI**.

Frontend harus menggunakan style:

```txt
Modern Neobrutalism
```

Tujuan style ini:

1. Membuat aplikasi terlihat bold dan memorable.
2. Tidak terlihat seperti admin template biasa.
3. Tetap friendly untuk user.
4. Tetap profesional untuk presentasi CEO.
5. Konsisten di seluruh halaman.
6. Mudah diimplementasikan dengan Tailwind CSS dan shadcn/ui.

---

# 2. Design Principle

## 2.1 Bold but Clear

UI boleh kuat secara visual, tetapi informasi tetap harus mudah dibaca.

## 2.2 Playful but Professional

UI boleh playful, tetapi jangan terlihat seperti game anak-anak.

## 2.3 Simple but Powerful

Komponen harus simple, tetapi mampu menampilkan data project yang kompleks.

## 2.4 Consistent Everywhere

Border, shadow, radius, spacing, dan warna harus konsisten di semua halaman.

## 2.5 Action-Oriented

Setiap card, button, dan alert harus membantu user memahami action berikutnya.

# 6. Core UI Components

## 6.1 BrutalButton

### Purpose

Reusable button untuk semua action utama.

### Variants

```txt
primary
secondary
success
warning
danger
ghost
```

### Sizes

```txt
sm
md
lg
icon
```

### Style Rules

Primary:

```txt
Background: Blue
Text: White
Border: 2px black
Shadow: 4px 4px black
```

Secondary:

```txt
Background: White
Text: Black
Border: 2px black
Shadow: 4px 4px black
```

Danger:

```txt
Background: Coral
Text: Black/White
Border: 2px black
Shadow: 4px 4px black
```

### Behavior

Hover:

```txt
Move -2px x/y
Shadow becomes 6px
```

Active:

```txt
Move 2px x/y
Shadow becomes 2px
```

### Props

```ts
type BrutalButtonProps = {
  variant?: "primary" | "secondary" | "success" | "warning" | "danger" | "ghost";
  size?: "sm" | "md" | "lg" | "icon";
  isLoading?: boolean;
  leftIcon?: React.ReactNode;
  rightIcon?: React.ReactNode;
  children: React.ReactNode;
};
```

### Usage Example

```tsx
<BrutalButton variant="primary" size="lg">
  Generate with AI
</BrutalButton>
```

